This tool provides a basic prediction of the difficulty of a given sentence in French, according to the CEFR proficiency levels (A1 being to easiest to understand and C2 the hardest). The prediction is based Google AutoML Natural Language machine learning model which was trained with 4800 sample sentences. The estimation is shown on the right side for each level, showing which one is the most likely to be correct. You can learn more about the project on the project's GitHub.
